Bluebox
===
Bluebox is a declarative components tree with inline styles experiment.

For now most of the work is being done on components and inline styles.

Features
---
- declarative
- inline styles
- renders to WebGL
- FlexBox layout system

Components
---
- Image [done]
- Text
- View [done]
